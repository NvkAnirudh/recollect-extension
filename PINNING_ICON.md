## Problem

## Alternative: Always Show Extension Icons

Some browsers let you show all extension icons by default:

### Chrome
- No global setting, must pin individually

### Arc
- Extensions automatically appear in the toolbar
- Can rearrange by dragging

### Brave

- Toggle on/off individual extensions

## Notes

- The Recollect icon will NOT appear "on the side" of pages
- It's a toolbar icon, not a floating page icon
- Different from extensions like Grammarly that overlay on the page
- This is standard Chrome extension behavior

## What the Icon Does

When clicked, it opens a popup showing:
- Current page URL (auto-detected on Instagram/YouTube/TikTok)
- "Save to Library" button
- Platform badge

## Mobile Note

Chrome extensions don't work on mobile browsers. For mobile:
- Use the bookmarklet instead (drag to bookmarks bar)
- Or visit /save page directly and paste URLs
